## sbt project compiled with Scala 3

### Usage

You can compile code with `sbt compile`
Run it with `sbt run`
Test it with `sbt test`
Start a Scala 3 REPL with `sbt console` you can then access when is defined in the program.
Clean generated temp files with `./clean.sh`


# New project

`sbt new scala/scala3.g8`
